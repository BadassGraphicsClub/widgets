# widgets

Public mirror of `froggy-os/Core/widgets` — deployed to https://badass-widgets.vercel.app via Vercel.

Do not edit here directly: the canonical source lives in the private froggy-os repo. Each change is edited there first, then mirrored to this repo (manual sync).

- `clock.html` — Paris time HH:MM:SS, Notion-embeddable. `?theme=dark` / `?theme=light` force the palette; no param = device auto-detect.
